# Windows 7 - Italian keyboard with tilde/backtick

Custom layout created using the
[Microsoft Keyboard Layout Creator](https://msdn.microsoft.com/en-us/globalization/keyboardlayouts.aspx).

## New combinations

* `` AltGr + ' => ` ``
* `AltGr + ì => ~`

## Installation

Download the repo as a zip, unzip it, run `bin/setup.exe`.
After that you'll find the new layout in the keyboard settings.
